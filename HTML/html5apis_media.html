HTML5 APIs & Media

1. What is the Geolocation API?

Geolocation API allows websites to get user's location (with permission). 

navigator.geolocation.getCurrentPosition(position => {
  console.log(position.coords.latitude);
});

Used in maps, delivery apps, ride booking systems.

2. What is Drag and Drop API?

Allows elements to be dragged and dropped. Common in dashboards and file uploads.

3. Difference between Canvas and SVG?

  • Canvas: Pixel-based, good for games & complex animations.
  • SVG: Vector-based, good for icons & scalable graphics.

4. What is Web Storage API?

  • localStorage: Persistent storage
  • sessionStorage: Clears after tab closes

5. What is Web Workers?

Allows JavaScript to run in background thread. Prevents UI blocking.

6. What is the difference between audio and video tags?

<video controls width="400">
  <source src="video.mp4" type="video/mp4">
</video>

HTML5 supports native media playback without plugins.
