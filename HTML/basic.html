BASIC HTML QUESTIONS

1. What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create and structure web pages. It defines the structure of content using elements like headings, paragraphs, images, links, and forms. HTML does not handle styling (CSS does) or logic (JavaScript does).

2. What is a Tag?

A tag is a keyword enclosed inside angle brackets (< >). It tells the browser how to display content. 

<p>Paragraph</p>

Here, <p> is the opening tag and </p> is the closing tag.

3. What is an Element?

An HTML element consists of an opening tag, content, and a closing tag. Everything from the start tag to the end tag is called an element. 

<p>Hello World</p>

4. What is an Attribute?

An attribute provides additional information about an element. Attributes are written inside the opening tag. 

<a href="https://example.com">Visit</a>

Here, href is an attribute that defines the link destination.

5. What is the basic structure of an HTML document?

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    Content goes here
  </body>
</html>

Every HTML document starts with DOCTYPE, followed by html, head, and body sections.

6. What is <!DOCTYPE html>?

It tells the browser that the document is written in HTML5. Without it, browsers may render the page in quirks mode.

7. What is the difference between block and inline elements?

Block elements take the full width available and start on a new line. Inline elements take only the necessary width and do not start on a new line.

  • Block: div, p, h1
  • Inline: span, a, img

8. What are heading tags?

HTML provides six heading tags from h1 to h6. h1 is the most important (used for main title), and h6 is the least important.

9. What is a paragraph tag?

The paragraph tag (<p>) is used to define a paragraph of text.

10. What is an anchor tag?

The anchor tag (<a>) is used to create hyperlinks. 

<a href="https://google.com">Google</a>

11. What is the img tag?

The image tag is used to display images on a webpage. It is a self-closing (void) element. 

<img src="image.jpg" alt="profile">

12. What is the difference between absolute and relative paths?

An absolute path includes the full URL. A relative path refers to a file location within the project.

13. What are lists in HTML?

Lists are used to group related items.

  • Ordered List (ol) – numbered
  • Unordered List (ul) – bullet points
  • List items (li)

14. What is a table?

A table is used to display data in rows and columns. 

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Sangeetha</td>
    <td>25</td>
  </tr>
</table>

15. What is a form?

A form is used to collect user input and send data to a server. 

<form>
  <input type="text">
  <button type="submit">Submit</button>
</form>

16. What are common input types?

  • text
  • password
  • email
  • number
  • radio
  • checkbox

17. What is the difference between id and class?

The id attribute must be unique within a page. The class attribute can be used multiple times.

18. What is the alt attribute in img?

The alt attribute provides alternative text if the image fails to load. It is important for accessibility and SEO.

19. What is the title tag?

The title tag defines the page title shown in the browser tab and search engine results.

20. What is the difference between HTML and HTML5?

HTML5 introduced new semantic elements (header, footer, section), audio and video support, new input types, and improved APIs.
