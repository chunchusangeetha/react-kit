CSS Advanced Interview Round

1. Difference between Flexbox and Grid?

Flexbox is designed for one-dimensional layouts (row or column). Grid is two-dimensional and controls both rows and columns.

Dashboards and complex layouts benefit more from Grid.

2. Explain flex-grow, flex-shrink, flex-basis.

flex-grow defines how much space an item can take. flex-shrink defines how it shrinks. flex-basis defines initial size before growing.

3. What is stacking context?

Stacking context determines how elements stack visually. It is created by properties like transform, opacity, and position with z-index.

4. What causes reflow?

Reflow occurs when layout changes (width, height, font-size). It forces the browser to recalculate positions.

5. What causes repaint?

Repaint occurs when visual properties change without affecting layout. Example: color or background change.

6. Why animate transform instead of width?

Transform animations use GPU acceleration. Animating width triggers layout recalculations.

7. What is CSS containment?

contain property limits layout and paint calculations. It improves performance in complex components.

8. What is clamp()?

clamp() allows responsive values without media queries. It defines minimum, preferred, and maximum values.

9. What is aspect-ratio?

aspect-ratio ensures consistent proportional resizing. Useful for images and video containers.

10. What are CSS variables?

CSS variables allow dynamic theming. They are defined using --name and accessed using var().

11. What are logical properties?

Logical properties like margin-inline support different writing modes. They improve internationalization.

12. What is object-fit?

object-fit controls how images scale inside containers. cover maintains aspect ratio while filling space.

13. What is pointer-events?

pointer-events controls mouse interaction. Used to disable clicks on overlays.

14. What is backdrop-filter?

backdrop-filter applies blur effects. Common in glassmorphism UI designs.

15. Explain BEM methodology.

BEM stands for Block, Element, Modifier. It improves maintainability in large projects.
